# Size
Kitchen comes with a few different sizes to choose from.

## Available Sizes

| Name | Size |
| :--- | :--- |
| `tiny` | 11px |
| `small` | 13px |
| `normal` | 16px |
| `medium` | 18px |
| `large` | 24px |
| `title` | 32px |
| `extraTitle` | 48px |

## Usage

You can use the `size` prop to change the size of the text.

```jsx
import { Text } from '@tonightpass/kitchen';

<Text size="large">This is a large text</Text>
```
