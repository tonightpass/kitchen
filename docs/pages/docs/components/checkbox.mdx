# Checkbox

Checkboxes allow the user to select one or more items from a set.

## Usage

First of all, you need to import the `Checkbox` component and the `useCheckbox` hook from the `@tonightpass/kitchen` package.

```js
import { Checkbox, useCheckbox } from "@tonightpass/kitchen"
```

## Default

<Playground
code={`() => {
  const [checked, toggle] = useCheckbox();

  return (
    <Container>
      <Checkbox label="Label" checked={checked} onChange={toggle}>
        Option
      </Checkbox>
    </Container>
  );
}`}
/>
